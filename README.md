# AI Resume Analyzer

## 1. Project Title
AI Resume Analyzer Using Generative AI

---

## 2. Introduction

AI Resume Analyzer is a web-based application that helps students and job seekers evaluate their resumes using Artificial Intelligence. The system extracts text from a PDF resume and uses Google's Gemini AI model to analyze the content. It provides a resume score, identifies skills, highlights strengths and weaknesses, and suggests improvements.

This project simplifies the resume review process and helps users create better resumes for placements and job applications.

---

## 3. Problem Statement

Many students and job seekers do not know whether their resumes are strong enough for job applications. Manual resume evaluation takes time and may not provide detailed feedback.

The AI Resume Analyzer solves this problem by automatically analyzing resumes and generating intelligent recommendations.

---

## 8. Modules

### Module 1: Resume Upload
Users upload their resume in PDF format.

### Module 2: Text Extraction
The application extracts text from the uploaded PDF.

### Module 3: AI Analysis
Gemini AI analyzes resume content.

### Module 4: Result Generation
The system generates:
- Resume Score
- Skills Identified
- Strengths
- Weaknesses
- Suggestions
- Suitable Job Roles

### Module 5: Display Output
Analysis results are shown on the web page.

---

## 9. Working Procedure

Step 1:
User uploads a PDF resume.

Step 2:
The system extracts text from the document.

Step 3:
The extracted text is sent to Gemini AI.

Step 4:
AI processes the resume and evaluates it.

Step 5:
The analysis report is displayed to the user.
